  • - Use semantic HTML to ensure readablity
  • - Use srcset on images to improve performance
  • - Launch and provide updates when needed
  • - Connect Firebase to use analytics

NHL Winners

After making this to get quick information on how the Canucks were doing, I decided to challenge myself to refactor the code and expand the radius to include all NHL teams. At a click of a button you can change the theme to your favourite teams colors, see when their next game is, check their record and even see who they play next. I used HTML, CSS, Javascript, Axios and Webpack to create this project

  • - Create a dynamic color theme based on user selections
  • - Pulled and displayed data from a NHL API
  • - Use Axios to call API and async and await to use the data
  • - Maintain an pleasent design on all team selections
  • - Organized Javascript using ES6 modules
